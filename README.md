# 뉴스파싱 및 요약

네이버 뉴스(주제별, 언론사별)를 파싱하고 KoBart 인공지능을 이용해 기사 내용을 요약한 뒤 Oracle Cloud에 데이터를 저장하는 파일

newsSummary 프로젝트의 데이터를 담당하며
Idx(항목구분), Title, Content, Img, Link 항목을 파싱하여 제공한다.

해당 프로그램은 1시간 간격으로 실행하여 DB 데이터를 갱신예정
